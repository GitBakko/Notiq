- `backend/src/utils/ydocIntegrity.ts` — `isDegenerateTipTapJson` (degenerate = empty doc or only-empty-paragraphs; image/table/hr-only is NOT degenerate). Used in `fetch()` (rebuild from content if ydoc degenerate over substantial content) + `store()`.
- `backend/src/services/sharing.service.ts` `updateSharedNoteContent` — closed the last open empty-overwrite hole on `PUT /share/notes/:id/content`.
- `NoteVersion` model + migration `20260610000000_add_note_version`. `backend/src/services/noteVersion.service.ts` — `snapshotPreviousVersion` (best-effort, throttle 2min, force on restore, retain 50-or-30d), `pruneNoteVersions`, `listNoteVersions`, `restoreNoteVersion`. Wired into updateNote + store() + updateSharedNoteContent (all best-effort, never block the primary save).
